## NPP / @oresoftware/npp

>
>  Publish your dependency trees to NPM with ease.
>  NPP goes hand-in-hand with NLU (npm-link-up)
>


### Caveats / Disclaimer

* Currently only works with Git
* You need Git version > 2.7.0 on your local machine


## Quick reference

>
>```bash
> npp view
>```
